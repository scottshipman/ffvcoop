ABOUT FFVCOOP
------------

FFVCOOP stands for Fresh Fruit and Veggie Coop. This is an ecommerce site to support this co-ops weekly fruit and veggie sales.

CONFIGURATION AND FEATURES
--------------------------

This site is managed via the features module

Week ending (pickup) dates can be setup in the Manage Main menu navigation for each corresponding year
Products are managed under the standard commerce module methods
